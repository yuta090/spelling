## 5. ハマりどころ（既知）
- **3つのパスワードは別物**：Supabaseログイン / API鍵(anon・service_role) / **DBパスワード**(Postgres)。混同注意。
- **DBパスワード画面は“表示”でなく“リセット”**。現行値は見られない。リセット→直後は再起動でConnection refused→数分待つ。
- **direct接続NG→poolerを使う**（上記）。
- **CLAUDE.md更新**：main直接コミット禁止／ブランチ必須／**実装後は必ずCodex Code Reviewerでレビュー**／指示外ファイルを触らない。
- `service_role` 鍵は**アプリに絶対入れない**（Edge/サーバーのみ）。anon と URL は埋め込み可。
- 子は匿名auth＋ペアリング。ペアリング/devices/membersへの**クライアント直接書込みは不可**（Edge経由）。
