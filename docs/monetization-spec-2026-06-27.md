# マネタイズ仕様 — フリーミアム設計（確定版 2026-06-27）

SpellingTrainer（日本人の子ども向け 英単語スペル練習 iPad アプリ）の課金モデル確定仕様。
関連: [HANDOFF-2026-06-26.md](./HANDOFF-2026-06-26.md)（StoreKit2 / entitlements TODO）, [srs-retention-design.md](./srs-retention-design.md)。

## 0. 大方針

- **モデル**：買い切りではなく **無料 + 月額サブスク（フリーミアム）**。
  - 子ども教育アプリは「**親が効果を実感してから課金**」が王道。買い切りは検討ハードルが高すぎる。
- **「買う人」は子どもではなく親**。CLAUDE.md の doer（子）/ manager（親）の軸をそのまま課金設計に適用する。
  - **子のエンゲージメントを作り、親の財布を開く**。2人を別々に攻略する。
- 課金フックは **2本立て（ハイブリッド）**：

  | ゲート | 何を出す/守るか | 誰を動かすか |
  |---|---|---|
  | **コンテンツゲート（レベル）** | 序盤レベルは無料、続きは課金 | **子**の「つぎやりたい」＝課金トリガー |
  | **機能ゲート（親）** | 採点・レポート・同期は課金 | **親**の「効果実感・ラク」＝払う理由 |

  → 子が壁を作り、親が払う理由を持ち、**1つのサブスクに収束**する。

## 1. 無料枠（フリーミアム / 期限なし）

ずっと無料で使える範囲。獲得と習慣化が役割。

| レーン | 中身 | 無料での扱い |
|---|---|---|
| **同梱の学年単語（Dolch）** | pre-K + K のプール（**92語** = pre-K 40 + K 52） | **92語まで無料解放**（Grade 1 以降はロック） |
| **親が手打ちで追加する単語** | 学校のテスト単語など | **別枠・実質無制限で無料**（総量上限なし） |
| **ローカル練習** | 端末内での練習・テスト | 無料 |
| **コイン / コスメ解放** | 練習で 3 コイン/回 → キャラ・背景解放（実装済み） | **無料のまま**（毎日来る理由） |

### 重要な確定事項
- **92語は「総量の上限」ではなく「同梱単語の無料解放ライン」**。
  - 1つの学習テスト（ステップ）に **Dolch の単語と手打ち単語を混ぜて組める**。子は「92語 + 自分の単語」を練習・テストできる。
- **手打ち登録は無料・無制限**（絞らない）。理由：
  - 「**学校のテスト単語を入れられる**」は最強の獲得フック。ここを頭打ちにすると一番刺さる体験を自分で潰す。
  - 手間という自然なゲートがあるため抜け道にならない。払う価値の本体は親機能側にある。
- 親の **単語登録・ステップ登録機能はそのまま無料**で使える。

## 2. 1日の新規語リズム（課金とは無関係）

- **1日に新しく出会う単語は最大 10 語**（上限。年齢で実質 3〜10 で可変運用可）。
  - 1語 × 3 ラウンド（`TestSettings.practiceRepetitions = 3`）。10語で約30書き＝子の集中の天井あたり。
- **登録元を問わず全単語に共通適用**（同梱でも手打ち学校単語でも 1日10語）。これは学習リズムであり課金ではない。
- 学習的にも正しい（分散学習）。「きょうのれんしゅう おわり！」を**コイン報酬で締める**（実装済み資産を流用）。
- 大量に登録されても毎日10ずつ消化されるだけで破綻しない。

### 無料の助走（効果実感が溜まる期間）
- 92語 ÷ 10語/日 ＝ **新規 約9日分**。復習を足して実質 **約2週間**。
- 約2週間で親ダッシュボードに効果実感（ストリーク・覚えた語数）が溜まり、壁に当たる頃が課金の好機。
- もし助走をもっと伸ばしたい場合のオプション：無料に Grade 1 を足す（→133語 = 約13日分）。**まず92語で出してデータを見て調整**。

## 3. 課金で開くもの（有料プラン）

### コンテンツ（子トリガー）
- Dolch **Grade 1 / 2 / 3 / noun** ＋ **NGSL 全バンド（約2,816語）** のレベルからの**自動セット生成**。
- 守るのは「**ラクして大量に**」の部分。手打ちは無料なので、課金が売るのは「ワンタップで学年/難易度セットを生成できる利便性 + 体系的な進行」。

### 親の機能（親トリガー）
- リモート採点（親がスマホから子の書き取りを採点）
- 学習レポート・分析（効果実感ダッシュボード）
- マルチデバイス同期（Supabase）

## 4. 価格・トライアル

| 項目 | 値 |
|---|---|
| 月額 | **¥580/月** |
| 年額 | **¥4,800/年**（実質2ヶ月分お得） |
| 無料トライアル | **7日間**（Apple Introductory Offer） |
| 価格画面の初期選択 | **年額**（実効単価が高く LTV が伸びる） |

### 「無料枠」と「7日間無料トライアル」は別概念（両立させる）

| | 無料枠（フリーミアム） | 7日間無料トライアル |
|---|---|---|
| 正体 | **ずっと無料**の体験 | **有料プランの“お試し”**（期間限定） |
| 中身 | 92語 + 手打ち登録 + ローカル練習 + コイン | 有料の全部（全単語 + 採点 + レポート + 同期） |
| 期限 | なし（永久） | 7日。過ぎると課金開始 or 無料枠に戻る |
| 仕組み | 自前のコンテンツ/機能ゲート | Apple の Introductory Offer（サブスクに付ける） |

**組み合わせ方（王道）：**
1. DL → **無料枠で遊ぶ（期限なし、92語）** ＝ 習慣化・効果実感
2. 壁に当たる（Grade 1 ロック or レポート見たい）→ **そこで7日トライアル開始**
3. 7日後 → 払えば有料、払わなければ**無料枠に戻るだけ**（92語はずっと使える）

→ **無料枠＝獲得と習慣化、トライアル＝購入の不安をゼロにする最後の一押し**。役割が違うので両方あると効く。

## 5. 子ども向け特有の地雷（遵守事項）

- 子に直接「買う」操作・価格・「有料」表記を見せない。**課金は必ず親ゲートの奥**（CLAUDE.md の UX 原則に合致。ストア審査でもNG）。
- 無料部分をケチりすぎない（熱が生まれず課金前に離脱する）。
- ガチャ的・射幸性のある課金を入れない（教育 × 子ども向けで致命的）。
- レベル（Dolch/CEFR/採点）等の専門用語を子の画面に出さない。

## 6. 集客（DL）メモ

- **ASO**：親が検索する語で最適化（「英語 子供」「スペル 練習」「英単語 小学生」）。スクショは親向けに「効果・記録・広告なし・安全」を訴求、子向けの可愛さは2枚目以降。
- **「広告なし・安全」を前面に**（日本の親に強く刺さる。子ども向けは広告と相性が悪い）。
- レビュー導線は、子がレベルクリアした**ポジティブな瞬間**に親へ依頼（親ゲートの奥）。
- 「おうち英語」「中学受験」界隈のインフルエンサー/ブログは費用対効果が高い。
